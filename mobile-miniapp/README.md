# Mobile Mini Program MVP

Minimal WeChat Mini Program front-end for the content workbench MVP.

Purpose:

- Capture inspiration on mobile.
- Submit it to the light cloud queue.
- Check sync status.

Default local cloud mock:

```text
http://127.0.0.1:8787
```

In WeChat DevTools, import this folder as a mini-program project. For local development, enable the DevTools option that skips domain validation, or replace the cloud base URL with a deployed HTTPS endpoint.

This mini-program intentionally does not run the full AI workflow. The desktop workbench remains the production surface.
