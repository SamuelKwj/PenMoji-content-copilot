# 抖音内容审稿 Skill

审稿流程：先检查限流红线，再做播放量诊断，最后给出修改建议。

## 输入
- 抖音视频脚本/文案/标题
- （可选）已发视频的后台数据：封面点击率、5秒完播率

## 第一步：限流红线检查（对照 2026 抖音新规）

### 四大限流类型 11 个检查点

#### 1. 暗线流（隐形坑）
- [ ] 涉及车辆 → 车牌号已遮挡？
- [ ] 没有出现任何商家地址/电话/联系方式（包括模糊处理）？
- [ ] 没有营销类词汇：招商、兼职、加盟、代理...？

#### 2. 零播放风险
- [ ] 没有蹭时政、未成年人、敏感人物热点？
- [ ] 没有炫富内容（豪车豪宅奢侈生活）？
- [ ] 没有擦边、低俗、恐怖画面？

#### 3. 明线限流（平台通知）
- [ ] 不是搬运/重复发布旧视频？
- [ ] **没有教学如何使用 AI 工具**（2026 严查）？
- [ ] 没有无资质讲医疗、中医、产品功效承诺？
- [ ] 没有明显利益诱惑引导？

#### 4. 算法限流
- [ ] 没有人为刷搜索流量？
- [ ] 预计搜索流量占比不会超过 25%？

#### 5. 2026 新增隐形雷区
- [ ] 不是批量 AI 生成低质内容，有个人原创观点？
- [ ] 不是同质化/模板化内容（一套文案多发/跟风换皮）？
- [ ] 视频中没有第三方水印/导流暗示？
- [ ] 没有未成年人不良导向内容？
- [ ] 没有长期抱怨/煽动对立/传播焦虑？
- [ ] 没有绝对化用词（保证、100%、最、第一...）？
- [ ] 没有暗示私域导流（进群、看主页、看简介）？

### 限流检查规则
- 只要有 1 项打叉 → 标记「**高风险**」，必须修改后再发
- 2-3 项边缘 → 标记「**中风险**」，建议调整
- 全中 → 标记「**合规**」，通过

## 第二步：播放量诊断（姜胡说三步法）

### 三个核心指标检查

#### 1. 封面/标题
- [ ] 标题让人一眼觉得「跟我有关」？
- [ ] 有没有痛点/冲突/悬念？

#### 2. 开头 5 秒
- [ ] 前三秒直接戳痛点？
- [ ] 有没有用「问题 + 解法」结构（你是不是XXX？我告诉你解决办法）？
- [ ] 封面和内容匹配吗？

#### 3. 内容密度
- [ ] 有没有废话/套话/背稿痕迹？
- [ ] 每 10 秒有没有一个核心信息点？
- [ ] 是否经过「口语化表达 → AI脱水」处理？

#### 4. 互动设计（2026 权重提升）
- [ ] 结尾有没有引导评论/提问？
- [ ] 内容有没有留下争议点/共鸣点引发讨论？

### 播放量诊断规则
- 如果已有数据：
  - 封面点击率 < 5% → 优先换封面
  - 5秒完播率 < 30% → 优先改开头
  - 整体完播低 → 砍废话提密度
  - 完播ok但推不动 → 加互动引导

## 第三步：输出结果
1. 限流风险评级（高/中/低/合规）
2. 列出问题项+具体修改建议
3. 播放量诊断问题+优化方案
4. 最终结论：可以发 / 需要修改后发

## 实操优化小贴士
1. 违规视频立刻**删除**，不要隐藏，隐藏不解除平台标记
2. 发完 3 小时内看数据，哪项指标不够，只改对应地方重新发，不用重拍
3. 2026 偏好原生真实内容，过度滤镜/包装反而会降权
4. 搜索流量自然增长最好，千万不要人为拉搜索，超过 25% 必被限
