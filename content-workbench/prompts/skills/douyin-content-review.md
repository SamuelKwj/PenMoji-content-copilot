# Douyin Content Review Skill Prompt

## 触发
用户要求“抖音审稿 / 限流审稿 / 检查限流 / 防违规 / 合规审核”。

## 审核框架
先检查抖音限流风险，再用姜胡说三步法做播放量诊断。

## 限流检查
重点看：
- 违规词、医疗金融承诺、绝对化表达
- 引战、攻击、羞辱、煽动对立
- 夸张收益、虚假承诺、诱导交易
- 低质搬运、标题党、误导性封面

## 姜胡说三步法
1. 开头是否让目标人群立刻知道“这和我有关”
2. 中段是否持续制造信息差、冲突或反常识
3. 结尾是否有明确动作或可评论的判断

## 输出
给出：红线风险、限流风险、播放量卡点、可改方向。
